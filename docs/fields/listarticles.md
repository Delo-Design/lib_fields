# Группа полей listarticles


## Поля

### Поле listarticles
#### Описание
Фиксированная выгрузка статей по указанным атрибутам для выбора.

#### Атрибуты
- category="1" - число | id категории материалов
- limit="15" - число | кол-во материалов
- published="0,1" - булево значение | статус публикации материала
- q="моя статья 1" - строка | фильтрация по имени статьи, внутри sql оборачивается как LIKE %<наша строка>%
- ids="1,2,3,4,5,6" - строка | через запятую id материалов, то есть покажет только с этими id 
- language="ru-RU" - строка | по какому языку искать

Унаследовано от поля List, поэтому частично доступны атрибуты. [Посмотреть можно тут](https://docs.joomla.org/List_form_field_type)

#### Пример xml
```xml
<field
    name="myarticle"
    type="listarticles"
    label="Мое название поля"
    description="Мое описание поля"
    published="1"
    limit="20"
    q="растение"
    addfieldpath="libraries/lib_fields/listarticles"
/>
```
